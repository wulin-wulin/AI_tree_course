# R007 - 知识森林地图蛇形折返排布

状态：Accepted

创建日期：2026-05-28

确认日期：2026-05-28

## 背景

森林地图的视觉（扁平几何树、宽幅风景、景深、配色）经 R004–R006 已被接受、保留。但**排布**始终不对：R005 是竖着长、R006 是横着长，都不是用户想要的。R007 只改**节点与小径的排布方式**，改成蛇形折返多行，使整幅匀称；其余视觉与功能保留。

本文件为可执行需求（`Ready`）。

## 用户真实反馈

> 原话保留：

- 整体挺好的，有一点不行，就是排布：之前是竖着长，现在（R006）是横着长。
- 我要的是弯弯曲曲走下来：第一行从左向右，然后拐下来从右往左，然后再从左往右，直到排布完毕。
- 不太宽不太长，均匀一点。

## 已确认的关键决策

- 只改**排布**，**保留** R006 的风景全景、景深、扁平几何树 icon、配色，以及 R004/R005 的结构、路由、数据与全部已有功能。
- 排布方式 = **蛇形折返多行（boustrophedon）**：一行排几个，到行末拐下来反向继续，逐行蛇形走完 8 个章节，顺序连续。
- 画面**匀称**：不太宽、不太长，接近均衡的版面；节点**间距均匀**。

## 本轮目标

把森林地图的 8 个章节节点与连接小径，从「单行横向铺开」改为**蛇形折返多行铺排**，让整幅地图匀称、节点分布均匀，同时保留 R006 已被认可的艺术质感。

示意（仅表达走向，非最终视觉）：

```
①→→②→→③
          ↓
⑥←←⑤←←④
↓
⑦→→⑧
```

## 具体需求

- **蛇形折返排布**：8 个章节节点分成若干行（如 3 行：3 / 3 / 2，具体行数与每行个数由执行按「匀称、不太宽不太长、间距均匀」自行取最佳）。
  - 第 1 行从左到右；行末用一段向下的弯曲小径**拐到第 2 行**；
  - 第 2 行从右到左；行末再向下拐到第 3 行；
  - 第 3 行从左到右……如此逐行折返，直到 8 个排完。
  - 序号 01–08 仍沿这条蛇形顺序连续、一眼可读；起点「林口」在序列开头、终点「林深处」在序列末尾。
- **连续弯曲小径**：小径把各行首尾平滑地连起来（行内横向走、行末弧形拐弯下行），整体是一条弯弯曲曲、连贯的路径，不是断开的几段。
- **匀称版面**：整幅接近均衡比例，不太宽也不太长；节点横向、纵向间距都尽量均匀、有呼吸感、不拥挤。
- **保留 R006 视觉**：扁平几何树 icon、配色、艺术质感保留。背景景物（天空/山丘/地面/装饰背景树等）**按蛇形多行版面相应调整**以保持协调——若单一侧视地平线不再适配多行铺排，可改为更贴合「地图/俯览」的统一扁平几何底景，但须保持与树一致的风格、配色克制、不堆砌、不拥挤。
- **保留全部已有功能（无回退）**：悬停/聚焦预览卡、进度点亮 + 「上次读到这里」高亮、章节配色、点树进入该章首个知识点（路由不变）、返回书架入口、可达性兜底（语义化 `<ol>` + 可聚焦 `<Link>` + `aria-label`，装饰图形 `aria-hidden`）。
- **响应式**：窄屏 / 移动端对蛇形多行版面优雅适配（可减少每行个数、增加行数，或等比缩放），保持匀称、可读、不拥挤。

## 非目标
