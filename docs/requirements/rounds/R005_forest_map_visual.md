## 本轮目标

在 R004 森林地图基础上，把树和小径重做成好看、统一、不呆板的视觉：树是一套扁平几何风格的精致 icon，小径是一条自然蜿蜒的徒步小径。整体审美达到「不再丑、不再呆板」的主观标准。

## 设计方向（防止再次翻车，执行须遵循）

### 树 icon —— 扁平几何 / 现代

- 一整套**风格统一**的扁平几何 icon：相同的边缘/圆角处理、相同的配色逻辑，8 棵树是同一视觉语言下的家族，不要每棵画风不同。
- 每棵树由**少量实心几何色块**构成（如 1–3 层三角松冠或圆润树冠 + 简洁树干），**平涂填色**；不要复杂渐变、不要多层细碎叶丛（上版正是这样显得粗糙）。
- 配色取该章 `cluster.accent` 为主色，辅以 `dark` 做单侧/底部的简单明暗，最多两三种色，保持扁平。
- **比例与轮廓**是好看的关键：树冠与树干比例协调、silhouette 简洁好认；避免上版的比例失衡与粗糙感。
- 可在统一风格内让树形有细微变化（例如树冠层数随该章知识点数量），但风格语言保持一致。

### 小径 —— 自然蜿蜒

- 一条**自然、略不规则**的蜿蜒小径（手绘 / 徒步地图感），从「林口」蜿蜒通向「林深处」。
- 8 个节点沿小径分布，**x/y 位置与间距都自然变化**，不是规则的左右交替锥形；**避免机械对称的之字形**（上版被否的正是这点）。
- 小径线条可用柔和描边 / 虚线 / 轻纹理表达「路」的质感，保持克制、不杂乱。
- 序号 01–08 与先后顺序仍要一眼可读；整体有序、不拥挤、可呼吸。
