## 已确认的关键决策

- 只做视觉/布局迭代，**不改** R004 的三级结构、路由行为、数据，**保留** R005 已可用的扁平几何树 icon 风格。
- 核心改动：① 地图**左右拉长**、用起横向幅面（解决「太窄」）；② 整体做成**宽幅风景全景**。

## 本轮目标

把森林地图从「窄的纵向蜿蜒」改为一幅**横向宽幅的风景全景**：小径自左（林口）向右（林深处）横跨整幅、用满横向空间，树与景物分布其上，有地面、远景与天空层次，像一幅有景深的森林风景画，而不再是几棵树挤在窄列里。

## 设计方向（宽幅风景全景，执行须遵循）

### 横向宽幅构图

- 地图采用**横向/全景幅面**，用满可用宽度，不再是窄的纵向列；桌面端整幅铺开。
- 小径自**左（林口）→ 右（林深处）**横向自然蜿蜒，8 个章节节点沿途分布，间距有疏密变化，序号 01–08 与先后顺序仍一眼可读。

### 风景层次（这是「艺术感」的来源）

- **地面 / 地平线**：树站在地面上，不再漂浮在白底；可有一条地面带 / 缓坡。
- **景深**：前景元素大、远景元素小；可加少量纯装饰的远近背景树（`aria-hidden`）营造森林纵深，但 8 棵**章节树**保持清晰可辨、尺寸适中、是视觉焦点。
- **远景山丘**：背后可有简洁的远山 / 起伏丘陵剪影。
- **柔和天空**：背景一层柔和天空（克制的渐变或柔色），营造氛围。

### 统一风格与克制

- 天空、山丘、地面、背景树都用**与树 icon 一致的扁平几何 / 现代风格**（平涂色块、干净、不堆砌细碎纹理），整幅是一套协调的视觉语言，不要变成粗糙插画。
- 背景配色**柔和、退后**，确保 8 棵章节树的 `accent` 配色仍突出、能承载「这是哪一章」。
- 「更艺术」来自构图、景深、层次与配色的整体协调，**不是堆装饰元素**；保持有序、不拥挤、可呼吸。

## 具体需求

- 重排 `ChapterMapPage` 的森林地图为横向宽幅风景全景：小径左→右横跨整幅、用满横向空间，替换当前太窄的纵向布局。
- 增加风景层次：地面/地平线、远景山丘、柔和天空、可选的远近装饰背景树（纯装饰、`aria-hidden`），营造景深与「画」的艺术感。
- 8 棵章节树沿小径横向分布，保留 R005 的扁平几何树 icon 风格（可按景深微调尺寸，但不推翻风格），配色仍突出可辨。
- 保留并继续可用：悬停/聚焦预览卡、进度点亮 + 「上次读到这里」高亮、章节配色、点树进入该章首个知识点（路由不变）、返回书架入口、可达性兜底（语义化 `<ol>` + 可聚焦 `<Link>` + `aria-label`，装饰图形 `aria-hidden`）。
- **响应式**：窄屏 / 移动端对宽幅全景做优雅适配（横向滚动浏览或等比缩放），保持可读、有序、不拥挤。
