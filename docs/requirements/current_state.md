## 当前项目状态

当前项目是一个《人工智能原理》课程知识可视化 demo，采用 Vite + React + TypeScript 实现，为纯前端单页应用。项目不接入后端、数据库或外部图片资源，课程内容和展示结构主要写在 `src/data/courseKnowledge.ts` 中。

当前页面整体由三个主要区域组成：

- 顶部课程工作台 Header：展示项目名称、课程展示定位、知识簇数量、知识点数量、动画示例数量和 Demo 状态。
- 主工作区左侧 Knowledge Atlas：以可视化坐标地图的形式展示 8 个知识簇和知识点节点，用户可以通过知识簇按钮、地图枢纽、地图节点和底部学习路径按钮切换当前知识点。
- 主工作区右侧知识点聚焦阅读页（R002 重构）：按“引子 → 核心思想 → 图示/动画 → 正文（原理、内联公式）→ 参考折叠区 → 延伸”的阅读层次组织内容，核心思想以大字导语呈现、图示/动画为版面中心；关键术语、相关对比、典型应用、前置知识默认折叠降权。面板顶部有阅读推进条，提供上一点/下一点（按全课程连续序列推进）及“{章节} · 第 x/n 点”位置感。

当前数据规模：

- 8 个知识簇。
- 49 个结构化知识点。
- 多种自绘 SVG 图示类型。
- 4 类轻量动画示例：注意力流动、梯度下降、A* 搜索扩展、智能体闭环。

当前主要交互：

- 点击知识簇筛选/展开地图区域。
- 点击地图中的知识点节点，右侧阅读页同步更新。
- 在知识点页用上一点/下一点连续推进阅读。
- “专注阅读 / 显示目录”开关：专注阅读时隐藏左侧章节导航，阅读页居中成单列占据主视野。
- 参考类次级内容（关键术语、对比、应用、前置知识）以折叠形式按需展开。
- 图示和动画作为知识解释内容展示，但用户对这些内容本身的交互仍较少。
