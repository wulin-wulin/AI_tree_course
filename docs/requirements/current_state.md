## 当前项目状态

当前项目是一个《人工智能原理》课程知识可视化 demo，采用 Vite + React + TypeScript 实现，为纯前端单页应用，使用 `react-router-dom`（HashRouter）做路由。项目不接入后端、数据库或外部图片资源，课程内容和展示结构主要写在 `src/data/courseKnowledge.ts` 中。

页面经 R003 重构为**三级递进导航**（HashRouter，URL 反映层级、刷新不丢位置、前进后退可用）：

- **Level 0 首页书架**（`/`）：标题「知识森林学习笔记」+ 4 张课程卡桌面端一字排开（单卡放大，窄屏自然回流为 2 列 / 1 列叠放）；仅「人工智能原理」可进入，其余三门为禁用占位态。整页内容垂直居中、首屏看得见全部 4 张卡，页面背景统一覆盖到底无白块。无 stat 计数器、无旧工作台 Header。
- **Level 1 章节导览**（`/ai`）：**知识森林地图**——一幅近 3:2 均衡比例的俯览扁平地图（柔和草地 + 林间空地斑块 + 散布装饰背景树），8 个章节作为扁平几何树以 **3/3/2 蛇形折返**铺排（行 1 左→右、行 2 右→左、行 3 左→右），一条 Catmull-Rom 平滑小径在行末弧形下拐、连贯穿过 8 棵树；序号 01–08、林口（序列开头）/ 林深处（末尾）清晰；每棵树取所属 cluster 的 accent 配色；通过 localStorage 标注「上次读到这里」并把走到该章为止的树标为「已长成 / 点亮」；悬停 / 聚焦树浮出预览卡（章节标题、副标题、梗概、知识点数）；提供「返回书架」入口；整幅用 CSS container queries（`cqw`）等比缩放，桌面与移动端是同一版面的不同尺寸。
- **Level 2 知识点阅读**（`/ai/:chapterId/:pointId`）：复用 R002 阅读分层（引子 → 核心思想 → 图/动画 → 正文/内联公式 → 参考折叠 → 延伸）；进入章节默认落在首个知识点；右侧悬浮章节列表展示本章知识点（可切换、可收起、不遮挡正文，桌面端为预留栅格列 + sticky，移动端为正文上方收纳条），并含切换章节 / 返回书架入口；上一点/下一点按全课程连续序列推进，章末自动进入下一章首点。

当前数据规模：

- 8 个章节（数据层为知识簇 cluster）。
- 49 个结构化知识点。
- 多种自绘 SVG 图示类型。
- 4 类轻量动画示例：注意力流动、梯度下降、A* 搜索扩展、智能体闭环。

当前主要交互：

- 首页选择课程（仅 AI 原理可进入）。
- 在森林地图上点击章节树，弹出本章预览方框（中央章节卡 + 椭圆环绕该章全部知识点小卡 + 虚线连接 + 「从头读起」CTA）；点中央 CTA 从该章首点开始读，点任一知识点小卡进入对应阅读页；ESC / 点方框外 / X / 点另一棵树都能正确关闭或切换；保留焦点陷阱与 ARIA。
- 阅读页用悬浮章节列表切换本章知识点，或用上一点/下一点连续推进（可跨章）。
- 参考类次级内容（关键术语、对比、应用、前置知识）以折叠形式按需展开。
- 三级之间有统一的页面级淡入转场。
- 图示和动画作为知识解释内容展示，但用户对这些内容本身的交互仍较少。
