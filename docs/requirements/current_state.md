## 当前项目状态

当前项目是一个《人工智能原理》课程知识可视化 demo，采用 Vite + React + TypeScript 实现，为纯前端单页应用，使用 `react-router-dom`（HashRouter）做路由。项目不接入后端、数据库或外部图片资源，课程内容和展示结构主要写在 `src/data/courseKnowledge.ts` 中。

页面经 R003 重构为**三级递进导航**（HashRouter，URL 反映层级、刷新不丢位置、前进后退可用）：

- **Level 0 首页书架**（`/`）：标题「知识森林学习笔记」+ 课程卡片列表；仅「人工智能原理」可进入，离散数学 / 编译原理 / 强化学习的数学原理为禁用占位态（无内容）。无 stat 计数器、无旧工作台 Header。
- **Level 1 章节导览**（`/ai`）：8 个章节连成一条带序号、连线的线性学习路径，能看出先后顺序；通过 localStorage 标注「上次读到这里」；可返回书架。
- **Level 2 知识点阅读**（`/ai/:chapterId/:pointId`）：复用 R002 阅读分层（引子 → 核心思想 → 图/动画 → 正文/内联公式 → 参考折叠 → 延伸）；进入章节默认落在首个知识点；右侧悬浮章节列表展示本章知识点（可切换、可收起、不遮挡正文，桌面端为预留栅格列 + sticky，移动端为正文上方收纳条），并含切换章节 / 返回书架入口；上一点/下一点按全课程连续序列推进，章末自动进入下一章首点。

当前数据规模：

- 8 个章节（数据层为知识簇 cluster）。
- 49 个结构化知识点。
- 多种自绘 SVG 图示类型。
- 4 类轻量动画示例：注意力流动、梯度下降、A* 搜索扩展、智能体闭环。

当前主要交互：

- 首页选择课程（仅 AI 原理可进入）。
- 章节路径上选择章节，进入该章首个知识点。
- 阅读页用悬浮章节列表切换本章知识点，或用上一点/下一点连续推进（可跨章）。
- 参考类次级内容（关键术语、对比、应用、前置知识）以折叠形式按需展开。
- 三级之间有统一的页面级淡入转场。
- 图示和动画作为知识解释内容展示，但用户对这些内容本身的交互仍较少。
